Zend_Http_Client_Adapter_Test

class Zend_Http_Client_Adapter_Test implements Zend_Http_Client_Adapter_Interface

A testing-purposes adapter.

Should be used to test all components that rely on ZendHttpClient, without actually performing an HTTP request. You should instantiate this object manually, and then set it as the client's adapter. Then, you can set the expected response using the setResponse() method.
