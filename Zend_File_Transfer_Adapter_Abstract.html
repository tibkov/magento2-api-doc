Zend_File_Transfer_Adapter_Abstract addPrefixPath(string $prefix, string $path, string $type = null)

Add prefix path for plugin loader

If no $type specified, assumes it is a base path for both filters and validators, and sets each according to the following rules: - filters: $prefix = $prefix . 'Filter' - validators: $prefix = $prefix . 'Validate'

Otherwise, the path prefix is set on the appropriate plugin loader.

string $prefix
string $path
string $type
