array getUrlDetails(string $url)

Get details on a particular bookmarked URL

Returned array contains four elements: - hash - md5 hash of URL - toptags - array of tags and their respective usage counts - url - URL for which details were returned - totalposts - number of users that have bookmarked URL

If URL hasen't been bookmarked null is returned.

Parameters

string $url URL for which to get details
