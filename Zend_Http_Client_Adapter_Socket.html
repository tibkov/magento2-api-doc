Zend_Http_Client_Adapter_Socket

class Zend_Http_Client_Adapter_Socket implements Zend_Http_Client_Adapter_Interface, Zend_Http_Client_Adapter_Stream

A sockets based (streamsocketclient) adapter class for ZendHttpClient. Can be used on almost every PHP environment, and does not require any special extensions.

Zend_Http_Client_Adapter_Socket setStreamContext(mixed $context)

Set the stream context for the TCP connection to the server

Can accept either a pre-existing stream context resource, or an array of stream options, similar to the options array passed to the streamcontextcreate() PHP function. In such case a new stream context will be created using the passed options.
