Zend_Db_Statement_Pdo_Oci

class Zend_Db_Statement_Pdo_Oci extends Zend_Db_Statement_Pdo

Proxy class to wrap a PDOStatement object for IBM Databases.

Matches the interface of PDOStatement. All methods simply proxy to the matching method in PDOStatement. PDOExceptions thrown by PDOStatement are re-thrown as ZendDbStatement_Exception.
