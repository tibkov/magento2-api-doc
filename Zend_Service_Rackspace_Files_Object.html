You must pass the ZendServiceRackspaceFiles object of the caller and an associative array with the keys "name", "container", "hash", "bytes", "contenttype", "lastmodified", "file" where: name= name of the object container= name of the container where the object is stored hash= the MD5 of the object's content bytes= size in bytes of the object's content contenttype= content type of the object's content last_modified= date of the last modified of the object content= content of the object
