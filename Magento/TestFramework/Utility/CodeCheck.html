Check if the specified function is called in the content.

Note that declarations are not considered.

If class context is not specified, invocation of all functions or methods (of any class) will be matched across the board

If some class is specified, only its methods will be matched as follows: - usage of class::method - usage of $this, self and static within the class and its descendants
