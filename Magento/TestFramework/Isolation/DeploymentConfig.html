void endTest(PHPUnit_Framework_TestCase $test)

Checks if deployment configuration has been changed by a test

Changing deployment configuration violates isolation between tests, so further tests may become broken. To fix this issue, find out why this test changes deployment configuration. If this is intentional, then it must be reverted to the previous state within the test. After that, the application needs to be wiped out and reinstalled.
