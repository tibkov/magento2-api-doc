ShipmentTrackInterface

interface ShipmentTrackInterface implements ExtensibleDataInterface

Shipment track interface.

A shipment is a delivery package that contains products. A shipment document accompanies the shipment. This document lists the products and their quantities in the delivery package. Merchants and customers can track shipments.
