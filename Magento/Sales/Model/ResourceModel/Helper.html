string escapeLikeValue(string $value, array $options = array())

Escapes value, that participates in LIKE, with '\' symbol.

Note: this func cannot be used on its own, because different RDMBS may use different default escape symbols, so you should either use addLikeEscape() to produce LIKE construction, or add escape symbol on your own.

By default escapes '_', '%' and '\' symbols. If some masking symbols must not be escaped, then you can set appropriate options in $options.

$options can contain following flags: - 'allowsymbolmask' - the '' symbol will not be escaped - 'allowstring_mask' - the '%' symbol will not be escaped - 'position' ('any', 'start', 'end') - expression will be formed so that $value will be found at position within string, by default when nothing set - string must be fully matched with $value
