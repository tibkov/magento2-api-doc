$this addColumn(string $name, string $type, string|int|array $size = null, array $options = array(), string $comment = null)

Adds column to table.

$options contains additional options for columns. Supported values are: - 'unsigned', for number types only. Default: FALSE. - 'precision', for numeric and decimal only. Default: taken from $size, if not set there then 0. - 'scale', for numeric and decimal only. Default: taken from $size, if not set there then 10. - 'default'. Default: not set. - 'nullable'. Default: TRUE. - 'primary', add column to primary index. Default: do not add. - 'primaryposition', only for column in primary index. Default: count of primary columns + 1. - 'identity' or 'autoincrement'. Default: FALSE.

Parameters

string $name the column name
string $type the column data type
string|int|array $size the column length
array $options array of additional options
string $comment column description
