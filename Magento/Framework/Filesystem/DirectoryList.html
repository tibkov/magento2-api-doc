DirectoryList

class DirectoryList

A list of directories

Each list item consists of: - a directory path in the filesystem - optionally, a URL path

This object is intended to be immutable (a "value object"). The defaults are pre-defined and can be modified only by inheritors of this class. Through the constructor, it is possible to inject custom paths or URL paths, but impossible to inject new types.
