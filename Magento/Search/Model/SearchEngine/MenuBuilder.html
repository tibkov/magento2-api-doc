MenuBuilder

class MenuBuilder

A plugin for Magento\Backend\Model\Menu\Builder class. Implements "after" for "getResult()".

The purpose of this plugin is to go through the menu tree and remove "Search Terms" menu item if the selected search engine does not support "synonyms" feature.
