Zend_View_Helper_Gravatar gravatar(string|null $email = '', null|array $options = array(), array $attribs = array())

Returns an avatar from gravatar's service.

$options may include the following: - 'imgsize' int height of img to return - 'defaultimg' string img to return if email adress has not found - 'rating' string rating parameter for avatar - 'secure' bool load from the SSL or Non-SSL location

Parameters

string|null $email Email adress.
null|array $options Options
array $attribs Attributes for image tag (title, alt etc.)

Return Value

Zend_View_Helper_Gravatar
