class PathTranslationParser implements Parser

translatePathMappings($mappings)

Given a list of path mappings, check if any of the targets are for directories that have been moved under the public directory. If so, update the target paths to include 'public/'. As no standard Magento path mappings should ever start with 'public/', and path mappings that already include the public directory should always have js/skin/media paths starting with 'public/', it should be safe to call multiple times on either.
