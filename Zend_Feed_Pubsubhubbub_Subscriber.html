Zend_Feed_Pubsubhubbub_Subscriber setPreferredVerificationMode(string $mode)

Set preferred verification mode (sync or async). By default, this Subscriber prefers synchronous verification, but does support asynchronous if that's the Hub Server's utilised mode.

ZendFeedPubsubhubbub_Subscriber will always send both modes, whose order of occurance in the parameter list determines this preference.

Parameters
