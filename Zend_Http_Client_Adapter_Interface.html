Zend_Http_Client_Adapter_Interface

interface Zend_Http_Client_Adapter_Interface

An interface description for ZendHttpClient_Adapter classes.

These classes are used as connectors for ZendHttpClient, performing the tasks of connecting, writing, reading and closing connection to the server.
