Zend_Tool_Project_Profile

class Zend_Tool_Project_Profile extends Zend_Tool_Project_Profile_Resource_Container

This class is the front most class for utilizing ZendToolProject

A profile is a hierarchical set of resources that keep track of items within a specific project.
